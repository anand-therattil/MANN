# MANN- Mental-Health Assessment using Neural Networks

Conversational AI for Mental health Evaluation triage 
+ this is an extention for the project of [Bolna Bhai](https://github.com/anand-therattil/Bolna_bhai)

## What It Does? 
It analyses the patient condition by having conversation with the AI agent and it will evaluate how urgent the patient need to be address by a health-care professional 

## How will it be shown 
We are trying to show the urgency through a dasboard which can be accessed in real time which would have the 
caller_number|priority
